== Description ==

Konfigurierbarer Webdesign-Newsticker fuer die Sidebar Deines Blogs. Es werden aktuelle Nachrichten aus dem Bereich Webdesign und Internet angezeigt. Konfigurierbar und anwendbar ueber ein Widget. Support: [Serian](http://www.serian.de/)

== Installation ==

1. Entpacke `webdesign-newsticker-german.1.0.zip`.
2. Lade den `webdesign-newsticker-german`-Ordner in Dein Verzeichnis `/wp-content/plugins` hoch.
3. Aktiviere das Plugin im 'Plugins'-Menue der Wordpress-Administration.
4. Fuege das Widget unter Design->Widgets deiner Sidebar hinzu und bearbeite die Widget-Optionen.
